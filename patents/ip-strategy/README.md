## 最大の差別化ポイント

### スマートレシート（東芝テック）との違い

> **東芝テック FAQ:**
> 「領収書については電子化はされず、今まで通り紙で出力されます」

| 観点 | スマートレシート | 本システム |
|------|-----------------|-----------|
| **対象** | レシート（買い物明細） | **領収書**（経費精算用） |
| **目的** | 家計管理・販促 | **経費精算・仕訳作成** |
| **勘定科目** | なし | **発行時点で付与** |
| **保管** | 利用者側のみ | **店舗・利用者双方で電帳法対応** |

### 既存経費精算システムとの違い

| 観点 | 既存システム | 本システム |
|------|-------------|-----------|
| **データ取得** | OCR（撮影/スキャン） | **店舗から直接** |
| **データ精度** | 90〜99%（OCR依存） | **100%（元データ直接）** |
| **勘定科目判定** | 受領後にAIが推定 | **発行時に店舗種別から自動付与** |
