## 2. 引用ネットワーク調査手順

### 2.1 Google Patentsでの調査方法

以下の手順でGoogle Patentsにて引用文献を確認してください：

#### ステップ1: 特許ページへのアクセス

各特許について以下のURLにアクセス：

```
東芝テック JP6598936B2:
https://patents.google.com/patent/JP6598936B2/ja

東芝テック JP6618585B2:
https://patents.google.com/patent/JP6618585B2/ja

freee JP5503795B2:
https://patents.google.com/patent/JP5503795B2/ja

freee JP5936284B2:
https://patents.google.com/patent/JP5936284B2/ja
```

#### ステップ2: 引用情報の確認

各特許ページで以下のセクションを確認：

| セクション | 英語表記 | 内容 |
|-----------|----------|------|
| **引用特許** | Citations | この特許が引用している先行技術 |
| **被引用特許** | Cited by | この特許を引用している後続特許 |
| **関連特許** | Similar | 技術的に類似する特許 |

#### ステップ3: 引用文献の分類

引用された文献を以下に分類：

| 分類 | 説明 | 活用方法 |
|------|------|----------|
| **特許文献** | 登録特許・公開公報 | 権利範囲の確認 |
| **非特許文献** | 学術論文・技術文書 | 公知例として活用可能 |
| **審査官引用** | 審査過程で審査官が引用 | 権利範囲の限定根拠 |
| **出願人引用** | 出願人が自ら引用 | 技術的背景の理解 |

## 6. 審査経過情報の確認

### 6.1 確認すべき内容

各特許について、J-PlatPatの経過情報から以下を確認：

| 情報種別 | 活用方法 |
|----------|---------|
| **拒絶理由通知** | 審査官が指摘した先行技術を把握 |
| **補正書** | 権利範囲が狭められた部分を確認 |
| **意見書** | 出願人の主張（権利範囲の限定解釈根拠） |
| **引用文献** | 審査官が引用した先行技術リスト |

### 6.2 J-PlatPatでの経過情報確認手順

```
1. J-PlatPat (https://www.j-platpat.inpit.go.jp/) にアクセス
2. 「特許・実用新案」→「特許・実用新案番号照会/OPD」を選択
3. 特許番号を入力（例: 6598936）
4. 検索結果から該当特許を選択
5. 「経過情報」タブをクリック
6. 「出願情報」「審査記録」を確認
7. 「書類一覧」から拒絶理由通知、意見書、補正書をダウンロード
```

## 付録C: 用語集

| 用語 | 説明 |
|------|------|
| **引用特許（Citations）** | ある特許が先行技術として引用している特許・文献 |
| **被引用特許（Cited by）** | ある特許を先行技術として引用している後続の特許 |
| **審査官引用** | 特許審査の過程で審査官が拒絶理由の根拠として引用した文献 |
| **出願人引用** | 出願人が出願時に先行技術として自ら引用した文献 |
| **公知例** | 特許出願前に公開されていた技術。無効審判の根拠となりうる |
| **経過情報** | 特許の審査経過（拒絶理由通知、補正書、意見書等）の記録 |
| **拒絶理由通知** | 審査官が特許要件を満たさないと判断した理由を通知する書面 |
| **補正書** | 拒絶理由通知に対して請求項等を修正する書面 |
| **意見書** | 拒絶理由通知に対して反論・説明する書面 |
