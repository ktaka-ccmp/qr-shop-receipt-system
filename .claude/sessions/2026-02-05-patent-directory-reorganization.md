## 方針変更: 請求項ベースの詳細FTO分析

### 新しいアプローチ

従来のレポート形式から、**弁理士が行うFTO調査の標準的な方法論**に変更：

| ステップ | 内容 |
|---------|------|
| **1** | 関連する特許番号・公開公報番号の一覧表を作成 |
| **2** | 各特許のドキュメントを作成し、請求項をすべて転記、メタ情報を記入 |
| **3** | 請求項ごとに構成要件を正確に理解 |
| **4** | ビジネスアイデアが権利侵害するかを請求項ごとに判定 |
| **5** | 回避すべき事項をドキュメントに追記 |
| **6** | サマリドキュメントを作成/更新 |

### 判定原則

> **オールエレメントルール**: 請求項の**すべての構成要件**を充足する場合にのみ侵害となる

## Key Decisions

### 1. ディレクトリ構成の分離

```
patents/
├── fto/                   # Freedom to Operate（侵害回避）
│   ├── summary.md
│   ├── avoidance-checklist.md
│   ├── expiry-timeline.md
│   └── reports/           # 8件の詳細レポート
│
├── ip-strategy/           # 自社IP戦略（権利化）
│   ├── invention-candidates.md
│   └── prior-art-searches/
│
└── archive/               # 過去の調査資料
```

### 2. CPM方式の認識修正

- 東芝テック特許6598936の核心は「ハイライト表示」であり、QRコードフロー（CPM/MPM）とは無関係
- CPM方式はDON'TからDO（CAUTION参照）に移動
- ただしAirレジ/スマレジの特許調査は未完了

### 3. 調査方法の発見

- **Google Patents** で日本特許の詳細情報（有効期限、分割出願等）が取得可能
- J-PlatPatへの直接アクセスは年金納付状況・審査書類閲覧時のみ必要
